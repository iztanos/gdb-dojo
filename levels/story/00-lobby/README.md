# Story Mode: Level 0 - The Lobby

The Lobby is the intro level for first-time GDB users.

It teaches the basic rhythm: read, build, run, inspect, and submit.

|---|---:|---|---|
| 00-welcome-desk | Available | build, run, submit | [00-welcome-desk](00-welcome-desk) |
| 01-glass-box | Available | start GDB, run, quit | [01-glass-box](01-glass-box) |
| 02-moving-door | Available | break main, run, continue | [02-moving-door](02-moving-door) |
| 03-seggy-fault | Available | info locals, print variable | [03-seggy-fault](03-seggy-fault) |
| 04-stamp-machine | Available | step, finish, return value | [04-stamp-machine](04-stamp-machine) |
| 05-lobby-exit | Planned | Level 0 capstone | - |
