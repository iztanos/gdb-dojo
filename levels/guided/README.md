# Guided Path

The Guided Path is the structured curriculum for people using GDB for the first time.

Exercises are ordered so that each one builds on previously introduced commands.

| Level | Focus | Status |
|---|---|---|
| [00 - Basics](00-basics) | Build, run, inspect variables, and step through functions | Complete |
| 01 - Breakpoints | Function and conditional breakpoints | Planned |
