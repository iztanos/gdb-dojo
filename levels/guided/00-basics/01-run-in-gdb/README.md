# 01: Run in GDB

## Skill

Start GDB, run a program, and quit GDB.

## Goal

Run the program from inside GDB and submit the status value it prints.

## Start

```bash
start
```

## Manual commands

```bash
make
gdb -q ./run-in-gdb
```

Inside GDB:

```gdb
run
quit
```

## Submit

```bash
./check.sh STATUS_VALUE
```
